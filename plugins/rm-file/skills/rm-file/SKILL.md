ファイルを削除する、またはファイルを上書きする際は、必ず`-f`オプションをつけてコマンドを実行してください。

## 実行ルール

### rmコマンド

ファイルを削除する時は、必ず`-f`オプションをつける。

```bash
rm -f <ファイル>
```

### cpコマンド

ファイルをコピーして既存ファイルを上書き削除する時は、必ず`-f`オプションをつける。

```bash
cp -f <コピー元> <コピー先>
```
